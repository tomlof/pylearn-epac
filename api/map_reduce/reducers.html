epac.map_reduce.reducers¶

Created on Thu Mar 14 16:12:26 2013

Reducers for EPAC @author: edouard.duchesnay@cea.fr

class epac.map_reduce.reducers.Reducer[source]¶

Reducer abstract class, called within the reduce method to process up-stream data flow of Result.

Inherited classes should implement reduce(result).

class epac.map_reduce.reducers.ClassificationReport(select_regexp='test', keep=False)[source]¶

Reducer that computes classification statistics.

select_regexp: srt
A string to select items (defaults “test”). It must match two items: “true/test” and “pred/test”.
keep: boolean
Should other items be kept (False) into summarized results. (default False)
>>> from epac import ClassificationReport
>>> reducer = ClassificationReport()
>>> reducer.reduce({'key': "SVC", 'y/test/pred': [0, 1, 1, 1], 'y/test/true': [0, 0, 1, 1]})
{'key': SVC, 'y/test/score_accuray': 0.75, 'y/test/score_precision': [ 1.          0.66666667], 'y/test/score_recall': [ 0.5  1. ], 'y/test/score_f1': [ 0.66666667  0.8       ], 'y/test/score_recall_mean': 0.75}
